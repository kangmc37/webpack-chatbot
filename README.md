# webpack-chatbot

- 一个webpack基础配置  
and
- 一个web版人工智障聊天机器人

## 技术栈
webpack + fetchAPI/axios + scss + ES6

## API
所使用Api来自[青云客智能聊天机器人API](http://api.qingyunke.com/)

## 运行
```
# 下载
git clone https://github.com/kangmc37/webpack-chatbot ./chatbot
cd ./chatbot

# 安装依赖
npm install

# 运行开发环境
npm run dev

# 构建生产环境
npm run build
```

浏览器中打开`localhost:8888`
